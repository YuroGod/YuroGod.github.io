# 关于小窝

# 为什么突然想起要搭建这么一个博客呢

# 一

我是一个有点懒的人，不太喜欢记笔记，奉行烂笔头不如好记性的理念
但随着年龄的增长，要学的东西越来越多，有些东西需要时常去回顾
因此准备在这里记录一些学习笔记，方便查阅和分享给有需要的人

# 二

也许生活并不总是那么的绚丽多彩，但也肯定有那么一瞬间是美丽的
美丽的事物就像昙花一样，只会出现在那一瞬间， 可能随着长大
伴随着来自各方面的压力，慢慢的会忘记那些曾经的美好
于是在这里记录一些生活中的小美好，也许在若干年后的某个夜晚
不经意打开这里，看到这些，也会让自己开心一下吧😀

# 三

不知道什么时候喜欢上了文学和哲学，时不时会有一些感悟
以及一些文学作品摘录，记录在此

# 四

在如此大的网络世界里，能有那么一块小小的角落属于自己
每次打开来，就像回到自己家一样，看着越来越多的笔记
自己也会有满满的成就感 hh

# 关于博客

  • 静态生成 Hexo

其实很早很早之前曾经用白嫖的阿里云服务器搭建过一个基于 wordpress 的博客
但是由于没钱去续费服务器和没有经验去应付莫名其妙的攻击和骚扰，于是在服务器过期后，那个网站也不了了之了

# Q&A

为什么网站加载这么慢，有时候进不去

因为该网站是部署在 Github 上的，需要一点点魔法啦～

为什么评论系统加载很慢，甚至有时候无法使用

博客的评论系统使用的 waline，部署在 Vercel 上，生成的 domains 速度很慢，有时候还会抽风打不开
后续考虑更换
